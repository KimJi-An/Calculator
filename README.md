# Calculator
Vanilla JavaScript로 계산기 만들기

- 입력한 수식과 계산한 결과를 표시
- 계산 결과가 소수라면 소수점 둘째 자리까지 반올림하여 표시
- AC 버튼 누르면 전체 초기화
- 괄호 우선 계산
